Chat Commands

nametag [option]
nametag chat command.

Displays nametag info or sets visibility.

Options:

  • hide : Sets player nametag hidden
  • show : Sets player nametag visible
  • status : Displays nametag text & visible state (default)

Usage:

     /nametag [option]
 /nametag hide
hidename
hidename chat command.

Sets player’s nametag hidden from others.

Usage:

    /hidename
showname
showname chat command.

Sets player’s nametag visible to others.

Usage:

    /showname
